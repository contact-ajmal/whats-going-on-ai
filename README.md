## üìñ About The Project

**WhatsGoingOnAI** is a futuristic, immersive platform designed to be the ultimate daily destination for AI enthusiasts, researchers, and engineers. It aggregates real-time intelligence from across the web, offering a curated stream of news, research papers, job listings, and educational resources.

The application features a distinct "Neural" aesthetic, utilizing WebGL-style particle animations, glassmorphism, and a recursive UI design system to create a premium, sci-fi user experience.
